EduEmp
Overview
EduEmp is a web-based Employee Management System designed specifically for educational institutions. It helps administrators efficiently manage staff data, roles, and workflows through a centralized and user-friendly platform.

 Features
 Employee Profile Management
 Institution-focused design
 Centralized employee database
  Add, update, and delete employee records
 Search and filter employees
 Secure authentication (if implemented)
 Organized and clean UI
 Tech Stack

Frontend:
HTML | CSS | JavaScript 

Backend:
Node.js 

Database:
MongoDB
